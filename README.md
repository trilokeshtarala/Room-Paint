# Web App: Wall Color Changer

Open `index.html` in your browser to run the app.

## Features
-   **Works on Mobile & Desktop**: Fully responsive.
-   **Touch Friendly**: Tap corners to select walls.
-   **Upload Your Own Image**: Click the `+` button to load a photo from your gallery/files.
-   **Categorized Colors**: Swipe through categories (Bedroom, Kitchen, etc.).
-   **Custom Picker**: Choose any color.

## Usage
1.  **Upload**: Click `+` to select an image.
2.  **Select**: Tap 4 corners of a wall. Tap "Finish".
3.  **Recolor**: Pick a category and tap a color!
4.  **Save**: Click "Save" to download the result.
